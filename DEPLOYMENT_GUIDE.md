## 🚀 How to Enable GitHub Pages

Follow these steps to publish your website on GitHub Pages:

### Step 1: Go to Repository Settings
1. Open your repository: https://github.com/Dearry12/CodingCamp-4May26-derrymeiraldy
2. Click on **Settings** tab (top right)

### Step 2: Navigate to Pages Section
1. In the left sidebar, scroll down and click on **Pages**

### Step 3: Configure GitHub Pages
1. Under **Source**, select **Deploy from a branch**
2. Under **Branch**:
   - Select `main` from the dropdown
   - Select `/ (root)` from the folder dropdown
   - Click **Save**

### Step 4: Wait for Deployment
1. GitHub will automatically build and deploy your site
2. This usually takes 1-3 minutes
3. Refresh the Pages settings page to see the deployment status

### Step 5: Access Your Website
Once deployed, your website will be available at:

**🌐 https://dearry12.github.io/CodingCamp-4May26-derrymeiraldy/**

## 🔧 Troubleshooting

### If the website doesn't load:
1. Check that GitHub Pages is enabled in Settings > Pages
2. Verify the branch is set to `main` and folder is `/ (root)`
3. Wait a few more minutes for the deployment to complete
4. Check the Actions tab for any deployment errors

### If styles are not loading:
1. Verify that `css/styles.css` is in the repository
2. Check browser console for any errors
3. Clear browser cache and reload

### If JavaScript is not working:
1. Verify that `js/app.js` is in the repository
2. Check browser console for any errors
3. Ensure the script tag in `index.html` is correct
